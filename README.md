# ENG
Alma is an offline text encryption tool that uses a user-defined key. The app requires no internet access, collects no data, and operates entirely locally.

### The app is designed for:

   private messaging;
   
   secure note storage;
   
   protecting links and code snippets;
   
   local text processing without sending data over the network.

### Key Features:

fully offline — no internet connection required;

no data collection — no analytics, trackers, or data transmission;

transparent — Android permissions do not include INTERNET or ACCESS_NETWORK_STATE;

user-defined key — only you know your decryption password;

modern encryption — **AES-GCM-256** with authentication.

### Technical Specifications:

Platform - Android

Minimum Version	- Android 7.0 (API 24)

Encryption Algorithm - AES-GCM-256

Operation Mode - Fully offline

Internet Permissions - None

# RUS
Alma — это простое и безопасное приложение для симметричного шифрования текста с использованием алгоритма AES-GCM-256. Ключ шифрования придумывает сам пользователь, приложение не хранит и не передаёт его.

### Приложение предназначено для:

конфиденциальной переписки **(само приложение НЕ является ни средством связи, ни организатором распространения информации)**;

безопасного хранения заметок;

защиты ссылок и фрагментов кода;

локальной обработки текста без передачи данных в сеть.

### Ключевые особенности:

полная автономность — работает без интернета;

отсутствие сбора данных — никакой аналитики, трекеров или передачи информации;

прозрачность — разрешения Android не включают INTERNET или ACCESS_NETWORK_STATE;

пользовательский ключ — только вы знаете пароль для расшифровки;

современный алгоритм — AES-GCM-256 с аутентификацией.

### Технические детали

Платформа	- Android

Минимальная версия - Android 7.0 (API 24)

Алгоритм шифрования -	**AES-GCM-256**

Режим работы - Полностью офлайн

Интернет-разрешения	- Отсутствуют
